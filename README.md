## Getting Started

---

- clone down the repo to your local machine
- create a .env.local file in the root of your project and create the following variables:

  - APP_SECRET
  - SALT_ROUNDS
  - DB_USERNAME
  - DB_PASSWORD
  - DB_NAME_TEST
  - DB_NAME
